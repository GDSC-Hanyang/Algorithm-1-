# 자연수 뒤집어 배열로 만들기
<div class="guide-section-description">
      <h6 class="guide-section-title">문제 설명</h6>

<ul>
<li>n은 10,000,000,000이하인 자연수입니다.</li>
</ul>

</tbody>
      </table></div>
